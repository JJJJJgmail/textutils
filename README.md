# textutils

A tiny collection of text helper functions, used as a practice project for my first pull request.

## Installation

No dependencies are required. Just clone the repo and import the module:

```python
from textutils import reverse_words

print(reverse_words("hello world"))  # "world hello"
```

## Available functions

- `reverse_words(text)` — reverse the order of words in a string.

## Running the tests

```bash
python -m pytest
```
